# FF14 跨区小助手 Pro

用于优化 FF14 相关网页登录与跨区操作体验，提供自动轮询并在可用时自动跨大区。

---

## 功能说明

### 1. 跨大区

- 支持自定义轮询时间间隔
- 支持在“整 10 分钟”时临时启用高速轮询（可选）
- 注：废物功能，但不想删

---

### 2. 自动确认用户需知

脚本现在可以自动点击“确认用户须知”，理论支持所有盛趣登录场景，包括：

- 商城登录（购买点卡 / 月卡）
- 网页跨大区登录
- 艾欧泽亚通行证登录
- 竞猜中心登录

---

## 使用方法

### 方法一：通过脚本管理器安装（推荐）

支持以下脚本管理器：

- Tampermonkey（油叉/叫油猴也行）
- ScriptCat（脚本猫）

安装步骤：

1. 打开脚本管理器
2. 新建脚本，或导入 `.js` 文件
3. 将本项目的 JS 文件内容粘贴进去
4. 保存并启用脚本

---

### 方法二：GreasyFork 一键安装

直接访问：

https://greasyfork.org/zh-CN/scripts/564602-ff14%E8%B7%A8%E5%8C%BA%E5%B0%8F%E5%8A%A9%E6%89%8Bpro

点击安装即可。
